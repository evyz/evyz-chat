# Мессенджер Evyz-chat
Мессенджер на NodeJs + ReactJs + Ws

Бек:
  - express
  - jwt-токены для авторизации
  - express-ws
  - psql DB
  - knex ORM

Фронт:
  - ReactJs,
  - Mobx и mobx-react-lite в качестве observer 
  - ws дефолтный
  - axios для доп. запросов на API бекенда
